# 典拠評価（evidence_grading）

## 3段階評価

### ✅ 確認済み（Confirmed）

以下のいずれかが揃っている:

- 書誌情報が複数の信頼できる DB で一致
- OCR テキストを画像（`page_image_url`）で目視確認済み
- DOI / CRID / NDL書誌ID などの永続 ID が確認済み
- 原文ページを `jp_lit_get_text_coordinates` で取得し内容を確認

### ⚡ 有力候補（Probable）

- 書誌情報は取得できたが現物未確認
- OCR 全文でヒットしたが目視確認なし（`jp_lit_search_fulltext` のみ）
- タイトル・著者・年が一致するが、同名別資料の可能性が排除できない
- レファ協の調査事例に挙がっているが一次情報で確認していない

### ❓ 弱い候補（Weak）

- タイトルや著者が類似しているがすべての書誌事項が一致しない
- LLM の推測による関連語でヒットしたが確認していない
- OCR のみで誤認識の疑いが高い（文字化け・断片）
- Web 上の二次情報のみ

## OCR 固有の注意

- OCR は誤認識を含む。特に以下は誤認識率が高い:
  - 変体仮名・くずし字
  - 旧字体・異体字
  - 縦組み・ルビ付きテキスト
  - 低解像度スキャン
- 重要な主張をする場合は **必ず `page_image_url` で画像目視確認** する
- `coordjson` の座標を使ってトリミング確認するとさらに精度が上がる

## 報告での使い方

```
結果を報告する際は必ず典拠強度を明示する:

✅ 確認済み: 「NDL 書誌 + デジコレ画像で確認しました」
⚡ 有力候補: 「CiNii に書誌情報はありますが現物未確認です」
❓ 弱い候補: 「タイトルが類似していますが別資料の可能性があります」

断定表現を使う際は「確認済み」のみとする。
「有力候補」以下では「〜の可能性がある」「〜と思われる」などを使う。
```
