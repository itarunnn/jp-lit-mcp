# 地方公共図書館・地域資料調査メモ

この文書は、地方人物、地方紙、地方雑誌、郷土史、自治体史、地域団体資料などを探すときに、公共図書館をどう調査経路へ組み込むかを整理する。`jp-lit-mcp` の固定 source 追加計画ではなく、`jp-lit-research` Skill が地域推定と検索計画を作るための運用メモである。

## 位置づけ

この領域は、一般に「地方公共図書館検索技術」ではなく、次の語彙で整理されている。

- 地域資料サービス
- 郷土資料
- 地方行政資料
- 公共図書館パスファインダー
- レファレンスサービス
- 新聞所蔵、雑誌所蔵、郷土人物

NDL リサーチ・ナビの「公共図書館パスファインダーリンク集」は、特定地域の情報は地域に根ざした公共図書館のほうが多く蓄積している場合がある、と説明している。公共図書館は、その地域で発行された図書・新聞・雑誌、地域行政資料、団体資料、ミニコミ誌、自費出版物、郷土人物資料を収集・保存する役割を持つため、NDL / CiNii / J-STAGE で薄い地方資料を補う導線になる。

## 根拠資料

- [NDL リサーチ・ナビ「公共図書館パスファインダーリンク集」](https://ndlsearch.ndl.go.jp/rnavi/plan/pubpath)
  - 都道府県立・政令指定都市立図書館のパスファインダーをまとめている。
  - 地域情報は地域公共図書館の蓄積が強い場合がある、という前提を示す。
- [NDL リサーチ・ナビ「さまざまな図書館の使い方」](https://ndlsearch.ndl.go.jp/rnavi/plan/post_1239)
  - 資料が決まっていない場合、公共図書館・大学図書館の書架、DB、パスファインダー、レファ協を使う流れを示す。
- [NDL リサーチ・ナビ「記事・論文の探し方」](https://ndlsearch.ndl.go.jp/rnavi/plan/post_605)
  - OPAC で論文・記事を探す場合、論文名ではなく雑誌名で検索し、巻号の所蔵を確認するのが有効とする。
- [NDL リサーチ・ナビ「新聞の所蔵機関を調べるには」](https://ndlsearch.ndl.go.jp/en/rnavi/newspapers/post_700037)
  - 地域の新聞所蔵は、都道府県・市町村立図書館の新聞所蔵一覧や地域ごとの包括的所蔵リストを確認する。
- [カレントアウェアネス「地域資料サービス」](https://current.ndl.go.jp/ca1846)
  - 地域資料サービスの研究動向、地域資料のデジタルアーカイブ化、レファレンス事例、公共図書館の役割を整理している。
- [文部科学省「公共図書館の特長」](https://www.mext.go.jp/a_menu/shougai/tosho/houkoku/05091401/005.htm)
  - 公共図書館の情報資産として、絶版資料、定期刊行物、地域行政資料、市民団体・NPO 発行のミニコミ誌、地域資料を挙げている。
- [カーリル for AI](https://calil.jp/ai/)
  - 2025年10月20日の更新履歴で、大学図書館および専門図書館のサポートを追加したと説明している。
- [カーリル図書館API仕様書](https://calil.jp/doc/api_ref.html)
  - 図書館カテゴリに `SPECIAL`（専門）が定義されている。
- 各県立図書館のパスファインダー
  - 栃木県立図書館、秋田県立図書館、山形県立図書館などは、郷土人物、新聞記事、地名、文学、方言などの調べ方を公開している。

## カーリルの扱い

[カーリル for AI](https://calil.jp/ai/) は Remote MCP であり、MCP endpoint は `https://mcp-beta.calil.jp/mcp`。公式説明では `search_books`、`search_libraries`、`get_ndc_info` の 3 tool があり、`search_books` は複数図書館システムを最大15館まで横断し、フリーワード、書名、著者、出版者、NDC分類、出版年で検索できる。

カーリル MCP は専門図書館も探索候補に入れられる。ただし専門図書館・資料室は、OPAC 非公開、閲覧制限、予約制、会員制、館内利用のみ、独自目録のみ、カーリル側で見つからない、といった場合がある。したがって、カーリル MCP の `search_libraries` で見つかれば `search_books` に入れ、見つからない場合は専門図書館自身の OPAC、目録、利用案内、レファレンスへ進む。

一方、[カーリル REST API](https://calil.jp/doc/api_ref.html) についての原則は「REST API はキーワード蔵書検索に使わない」。REST API の蔵書検索は `ISBN + systemid` が前提であり、人物名、地方紙名、地方雑誌名、地名、団体名から資料を発見する用途には足りない。REST API は ISBN 既知の所蔵確認や図書館データベース確認に限る。

したがって、この用途でカーリルを使う場合は、カーリル Remote MCP を外部 MCP として併用する。`jp-lit-mcp` 側は地域推定、検索語設計、見るべき館の優先順位づけ、結果評価を担当し、カーリル MCP は公共図書館 OPAC の実検索を担当する。

カーリル Remote MCP は、REST API のようなアプリケーションキー登録ではなく、初回 OAuth 認可で接続する。開発 checkout で live 接続を確認する場合は `npm run smoke:calil-mcp` を実行する。初回実行時は表示された URL をブラウザで開いて許可する。ブラウザを自動で開く場合は `CALIL_MCP_OPEN_BROWSER=1 npm run smoke:calil-mcp`、蔵書検索を省いて図書館検索だけ確認する場合は `CALIL_MCP_SKIP_BOOK_SEARCH=1 npm run smoke:calil-mcp` を使う。API キー登録は不要。

## 発動条件

次のいずれかに当てはまる場合、地方公共図書館ルートを検討する。

- 地方人物、郷土人物、出身者、在住者、地域ゆかりの人物を探している。
- 地方紙、地域紙、地方雑誌、ミニコミ誌、自治体広報、団体機関誌を探している。
- 地方史、市町村史、県史、郷土史、地域文化、祭礼、方言、地名、家系、地域団体が主題。
- NDL / CiNii / J-STAGE / IRDB では書誌・論文が薄く、地域の資料所蔵や地域レファレンスが重要そう。
- Japan Search やレファ協で地域アーカイブ、県立図書館、市町村立図書館、郷土資料室が手がかりとして出た。

人物名があるだけでは地方公共図書館ルートを発動しない。有名人・全国的作家・研究者・政治家などは、まず NDL / CiNii / J-STAGE / IRDB / Japan Search / 典拠・専門 DB / 主要新聞 DB を優先する。地域ルートは、出身地・居住地・活動地・没地・勤務先・事件地・媒体発行地などの地域性が調査目的に関係する場合、全国系 DB が薄い場合、または郷土人物・地域報道・地域資料・記念館・文学館・資料室が有力な手がかりとして出た場合に追加する。

地域性とは別に、人物の専門領域が強い場合は専門資料機関ルートを検討する。これは特定分野の「専門図書館」だけを探すのではなく、専門領域の博物館、資料室、アーカイブ、協会、学会、研究会、業界団体、同人団体、機関誌、会報、展覧会図録、個人コレクション公開情報を含める。分野名そのものの専門図書館が見つかりにくい領域でも、協会史、研究会誌、専門誌、博物館・資料室、関係団体の刊行物が文献発見の入口になる。

## 地域推定の材料

人物名だけでなく地名・媒体名・団体名・発行地を分けて集める。

- 出身地、出生地、没地、居住地、活動地、赴任地、勤務先所在地
- 作品・事件・団体・学校・寺社・企業・新聞社の所在地
- 地方紙・地方雑誌の発行地、旧題、前身紙、後継紙
- 旧国名、郡名、旧市町村名、合併前自治体名、通称地名
- NDC や件名から見た地域分類、郷土分類記号

地域候補は `pref`、`city`、旧地名、活動地域、発行地に分ける。範囲は狭ければよいわけではない。県立図書館は地域資料の基準点として外さない。県域全体の郷土資料、地方紙、県人録、市町村史、行政資料を持つ可能性が高いからである。市区町村中央館、地域図書館ネットワーク、郷土資料室、専門図書館・資料室は、県立図書館に加えて組み合わせる。

カーリル MCP の最大15館制限に対しては、次の順で候補を組む。小さい館へ一直線に狭めるのではなく、県立図書館と局所館を組み合わせる。

1. 該当都道府県立図書館
2. 該当市区町村中央館
3. 県内/広域の図書館ネットワーク
4. 発行地・活動地に対応する中央館
5. 郷土資料室・分館
6. 隣接自治体や旧郡域の館
7. 専門図書館・資料室（カーリルで見つかる場合）

## 検索順序

1. レファ協・NDL リサーチ・ナビで調べ方を確認する。
2. NDL / Japan Search / Web NDL Authorities などで人物・媒体・団体・事件から地域候補を抽出する。ただし人物名だけで地域公共図書館へ進まず、地域性が調査目的に関係するかを確認する。
3. 人物の専門領域が強い場合は、分野名から専門資料機関、博物館、資料室、アーカイブ、協会、学会、研究会、機関誌、会報、専門誌を探す。専門館名が既知でなくても、`人物名 + 分野名`、`分野名 + 資料室`、`分野名 + 協会 OR 研究会`、`分野名 + 機関誌 OR 会報` を試す。
4. Web 検索で、地域候補に対応する実在の図書館名、地域パスファインダー、新聞・雑誌所蔵一覧、郷土資料ページを拾う。人名は、地域性が必要な場合に限って、まず出身地・居住地・活動地・郷土人物としての地域を割り、そこから地域図書館へ進む。文学館、記念館、資料館、専門図書館、資料室への直接導線は補助として探す。
5. 地域候補ごとに、該当都道府県立図書館を基準点として外さない。そのうえで、該当市区町村中央館、県内/広域の図書館ネットワーク、発行地・活動地に対応する中央館、郷土資料室・分館、隣接自治体や旧郡域の館、専門図書館・資料室を追加候補にする。
6. カーリル MCP の `search_libraries` では、機械生成した館名だけに頼らず、Web 検索で見つけた実在の館名・ネットワーク名・資料室名を優先して `systemid` を取得する。
7. カーリル MCP の `search_books` で、最大15館の範囲に絞って検索する。
8. 検索語は資料種別ごとに変える。
9. ヒットしない場合は、地域パスファインダー、新聞所蔵一覧、レファ協事例、図書館レファレンスへの相談導線、専門団体・資料室への照会導線を次アクションに残す。

専門図書館・資料室が効きそうな例:

- 新聞博物館・新聞ライブラリー系
- 文学館・郷土人物記念館の資料室
- 博物館・文書館・地域資料センター
- 業界団体・企業史料室
- 大学の地域研究センター / 附属図書館
- 県史編さん室・自治体史料室系

地域候補と検索語を手早く構造化したい場合は、Skill 同梱の `scripts/plan-regional-library-search.mjs` に JSON を渡して、Web 検索 query とカーリル MCP 用の `search_libraries` / `search_books` 計画を作る。これはカーリルを直接呼ばない補助スクリプトである。人名だけでは地方人物扱いにせず、人物名から出身地・ゆかり地域を確認する query を出すに留める。地域候補や地方人物 topic が与えられた場合に、地域図書館へつなぐ下書きとして使う。専門領域が強い場合は `subjectKeywords` を渡し、専門館名が未確定でも、専門資料機関・協会・研究会・機関誌・会報への Web 検索導線を作る。媒体名・団体名からは所蔵館・資料室の手がかりも拾い、最大15館以内へ絞るための下書きとして使う。

```powershell
@'
{
  "personNames": ["山田太郎"],
  "placeNames": ["岐阜県中津川市"],
  "mediaNames": ["東濃新報"],
  "publicationPlaces": ["中津川"],
  "topics": ["地方人物", "地方紙"]
}
'@ | node skills/jp-lit-research/scripts/plan-regional-library-search.mjs
```

## 検索語の作り方

図書・郷土人物資料:

- 人物名
- 人物名 + 地名
- 人物名 + 職業・号・別名義
- 地名 + 人物
- 県人録、人物事典、郷土人物、伝記、顕彰、追悼

地方紙・地方雑誌:

- 地方紙・地方雑誌は記事名より媒体名で探す。
- 媒体名、旧題、前身紙、後継紙、発行地、新聞社名を試す。
- OPAC では記事名が入っていない場合があるため、巻号情報・所蔵範囲・欠号を確認する。

郷土史・自治体資料:

- 地名 + 史、誌、沿革、年表、町史、市史、県史、郷土史
- 旧地名 + 現地名
- 団体名 + 記念誌、沿革史、会報
- 学校名・企業名・寺社名 + 記念誌

## 結果評価

- カーリル MCP のヒットは「公共図書館 OPAC の候補」であり、書誌同定の最終確認ではない。
- NDL / CiNii で同一書誌が見つかる場合は、書誌情報を照合する。
- 地方紙・地方雑誌は、記事本文ではなく媒体所蔵の発見に留まることが多い。
- 所蔵範囲、欠号、閲覧条件、複写条件、館内限定、郷土資料室配架は必ず確認事項として残す。
- 「その地域の図書館にしかない可能性」は強い仮説だが、実在確認は公式 OPAC、図書館レファレンス、現物確認で行う。

## Skill への落とし込み

Skill は次を行う。

- 地方人物・地方媒体・郷土資料の兆候を検出する。
- 地域候補と検索語候補を分けて作る。
- `ndl_search` / `japan_search` / レファ協 / NDL リサーチ・ナビで地域と媒体を確認する。
- Web 検索で、実在する地域図書館、地域パスファインダー、新聞・雑誌所蔵一覧、郷土資料ページを拾う。人名は地域性が必要な場合に地域を割ってから地域図書館へ進め、文学館・記念館・資料館は補助導線として拾う。専門領域が強い場合は、分野名から専門資料機関・協会・研究会・機関誌・会報も拾う。媒体名・団体名からは所蔵館や資料室も拾う。
- カーリル Remote MCP が使える環境では、Web 検索で得た館名を `search_libraries` に渡し、15 館以内の `systemid` 群へ絞って `search_books` へ渡す。
- カーリル Remote MCP が使えない環境では、地域パスファインダー、図書館 OPAC、新聞所蔵一覧、レファレンス相談を次アクションとして返す。

固定 source へ混ぜない。地域公共図書館ルートは、調査の途中で地域性が強く出た場合に発動する補助経路として扱う。
