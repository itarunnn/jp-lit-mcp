# 検索語展開（query_expansion）

## 原則

1. 展開した語はすべて「候補」として扱う
2. 実際にヒットした語だけを採用する（LLM 推測のみで断定しない）
3. 展開語でヒットがない場合は「試したが見つからなかった語」として報告に含める

---

## 展開カテゴリ

### A. カタカナ外来語の揺れ

末尾長音省略・濁点変異・母音変異のパターンを試す。

```
例: ラジオ
  → ラヂオ（歴史的表記）
  → ラジヲ（旧仮名遣い寄りの表記）
  → 無線電話（当時の説明的表現）
  → 放送（関連する上位語）
```

外来語が入ってきた時期が明治期なら、ドイツ語・フランス語由来の発音表記も候補に加える。

### B. 旧字体・新字体

現代常用漢字で検索して足りない場合、旧字体を試す。

```
例: 亜細亜 ↔ 亞細亞
    国会 ↔ 國會
    図書 ↔ 圖書
    芸術 ↔ 藝術
```

NIHU Bridge は `normalize=true`（デフォルト）で異体字同定するため、旧字体の試行が最も有効。

### C. 旧仮名遣い

```
例: 言う → いふ
    思う → おもふ
    会う → あふ
    ウィルス → ヰルス / ヸルス
```

### D. 漢語・翻訳語

明治期には外来概念に漢語訳があることが多い。

```
例: 野球 → ベースボール（逆方向も）
    電信 → テレグラフ
    民主主義 → デモクラシー / 民主々義
    ストライキ → 盟休 / 罷業
```

### E. 上位語・下位語・同義語

```
上位語: 「職業婦人」→ 婦人問題 / 労働問題 / 女子教育
同義語: 「選挙」→ 投票 / 票決 / 公選
下位語（ノイズに注意）: 「花」→ 桜 / 菊
```

### F. 人物名の揺れ

```
表記: 樋口一葉 / 樋口奈津 / 樋口夏子
号・通称: 幸田露伴 / 幸田成行 / 露伴
```

### G. 典拠・分類由来の語

Web NDL Authorities 由来の語は、出どころごとに強さを分ける。

```
人名 variant_terms:
  シェイクスピア / Shakespeare / 寒格斯比亜

人名 same_identity_terms:
  色川武大 / 阿佐田哲也 / 井上志摩夫

著作 variant_terms:
  Le petit prince / 星の王子さま / 小さい王子

分類由来:
  NDC 596.7 → 飲料 / コーヒー / 茶 / 酒 など
```

- `variant_terms`: 原則として検索語候補にしてよい
- `same_identity_terms`: 同一人物・同一団体の別名義。まとめて探すか、名義別に探すかを分ける
- `reference_terms`: 上位語・下位語・関連語。検索範囲を広げるので慎重に使う
- 分類由来語: 未知文献探索の入口。分類範囲が広い場合は年代・資料種別・source を絞る

## 時代別の重点展開

| 時代 | 優先する展開 |
|------|------------|
| 江戸・幕末 | 旧字体 / 変体仮名 / 漢籍由来語 |
| 明治 | 旧字体 / 旧仮名遣い / 漢語翻訳語 / 擬洋風表記 |
| 大正・昭和戦前 | カナ外来語揺れ / 旧字体 / イデオロギー用語 |
| 昭和戦後 | 新旧字体混在 / 外来語揺れ |
| 現代 | 同義語 / 専門用語 / 略語 |

---

## 実施ステップ

```
1. 元の検索語でベースラインの検索
2. 展開候補を列挙（カテゴリ A〜G から選択。必要なら `jp_lit_resolve_authority` / `jp_lit_find_authority_terms_by_classification` を使う）
3. 展開語を1〜2語ずつ検索
4. ヒットがあった語を「採用語」とする
5. ヒットなしの語を「試行済み・不採用」として記録
6. 採用語で改めて検索し結果を統合
```

---

## 注意点

- LLM が推測した展開語を、検索確認なしに「この語でヒットした資料がある」と言ってはいけない
- 展開語の候補が多すぎる場合は、時代・文脈的に最も可能性が高いものから試す
- NIHU Bridge の `normalize=true` で解決できる場合は個別展開の前に試す
