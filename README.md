# ndl-jp-lit-mcp

NDL Search と NDLデジタルコレクションを対象にした日本語文献探索向け MCP サーバー。

## 開発

```bash
npm install
```

Task 1 はプロジェクト初期化のみを対象にしており、依存解決まで完了しています。

- `npm install` は実行可能です。
- `npm test` はテストファイル未作成のため、Task 1 時点ではまだ成功しません。
- `npm run dev` は `src/index.ts` 未作成のため、Task 1 時点ではまだ起動しません。
- `npm run build` も同様に、アプリ本体の `src/` 実装追加後に利用する想定です。
